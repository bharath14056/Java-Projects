## 🧠 OOP Interview Preparation Notes

### 📌 Class vs Object
- Class: Blueprint (Method Area)
- Object: Runtime entity (Heap Memory)

### 📌 Object Creation Process
When `new Student()` is executed:
1. Class loading verification
2. Heap memory allocation
3. Default value initialization
4. Constructor execution
5. Reference assignment

### 📌 Constructor Insights
- Java provides default constructor if none is defined
- Private constructors are used in Singleton design pattern
